Before and after image slider block built with JuxtaposeJS.

== Description ==

Just provide two images and Juxtapose will do the rest of the work for you.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/juxtaposejs-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
